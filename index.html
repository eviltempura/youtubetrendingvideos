Introduction¶

In this tutorial I will walk you through a brief introduction on what data science is and how Python is used to perform data analysis, and more specifically I will be showing you how to use some of the basic methods and packages to analyze dataset.

1. Preparation¶

1.1 Use the right libraries¶

One of the reasons of why Python has been gainning its popularity is that it has a huge set of libraries ready to be used. Unlike Java or C/C++, with the help of these libraries, Python is a very "efficient" language, and by "efficient" I do not mean that Python runs faster, but that it is easy to use and thus facilitates the work. The libraries that I am going to use in this tutorial are as listed below with a short description.

NumPy: A powerful and almost necessary package for scientific studies and data analysis, it provides a set of methods that allow users to manipulate N-dimensional arrary objects.

Pandas: Pandas supports a set of data structures including tables and series, and it works closely with NumPy. Pandas is heavily used to store and process formatted data (e.g., csv files, json files html tables and etc.)

Matplotlib: Matplotlib offers a plotting function that can is mainly used for data visualization.

1.2 Data collection¶

There are two ways of getting data. You can either scrape data from websites and other data sources, or you can download datasets from online platforms like Kaggle. Both of these methods have their advantages and disadvantages, and how to choose the proper way of collecting data is crucial to data science.

Scraping raw data is a more timely-manner method compared to downloading datasets, because online datasets are often collected a long time ago and may not be able to correctly represent the problem in time. The update of online datasets is utterly up to the uploader and sometimes the uploader would stop updating for various reasons. Although online datasets could be collected long before, there are some exceptions. If the source from which the online dataset is collected has stopped updating or has a long cycle of updating, then the dataset can be seen as up to date. For example, a dataset of World War II is always up to date, becuase the war is already finished and the event itself will never generate new data.

Using ready-to-go datasets can save a lot of time because data scraping is a very time-consuming process. Scraping raw data involves locating the source of data, collecting the data and tidying the data. The time spent on each of these three steps can vary greatly and is largely determined by the format and the tidiness of the raw data. On the other hands, the online datasets are easy to use because they are already tidyed by the data miners and are put into easily accessible formats (e.g., csv files and character delimited text files).

In this tutorial, I will be using a dataset available on Kaggle (https://www.kaggle.com/datasnaek/youtube-new)

3.1 Total views vs. date¶

3.1.1 A simple plot¶

Now, we have a table with everything formatted to our needs, the next step is to find out the total views in each day.

First, we list out all the different dates in which the trending videos were recorded. This is done with Pandas unique() function which returns a set of all the values in a column.

Then, all the views within one day were summed together with the aggregate function sum(), and by going through each element in dates, the total views of all the trending vidoes in each day were put into a list called views

Last, we convrt the unit of views into million.

In [303]:

3.1.2 What do we find¶

Looking at fig.1, we can easily find that there is a wave-like pattern in the views over time. Now let's assume that this pattern is not random but instead it has a reason. To find out the reason to this pattern we start with making guesses, and let's guess this pattern is related to what day it is in a week. We need to verify if our guess is correct, so let's highlight weekends and see if it matches the pattern.

4.1 Regression¶

Originally developed in statistics, the method of regression has been borrowed and heavily used in the field of machine learning. I am going to demonstrate how to use linear regression to find out the relationship between number of likes and number of views.

The regression is generated with Numpy's poly1d() function.
